# Travel Blog React App

A simple React application demonstrating core travel blog features.

## Features

- Display of travel locations on a map. (Implemented with React Leaflet)
- Sample blog post display.
- Basic UI elements for comments and subscriptions.
- Minimalist design with a blue and white color scheme.

## Instructions

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the application: `npm start`
4. View the application in your browser at `http://localhost:3000`

## Dependencies

- React
- React Leaflet
- Leaflet
